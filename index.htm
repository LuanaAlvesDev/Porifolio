Sou estudante e entusiasta de análise de dados e desenvolvimento web, em transição de carreira e em constante aprendizado. Tenho me dedicado a adquirir conhecimentos sólidos em Python, SQL, Power BI e Excel para análise de dados, além de HTML, CSS, JavaScript e ferramentas como React para o desenvolvimento de aplicações web.
